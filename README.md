# PHP for WordPress

This repository contains reusable PHP utilities, filters, and actions
commonly used in WordPress development.

The focus is on clean code, performance, and proper usage of
WordPress hooks and core APIs.

## Included Examples

### Filters
- Custom excerpt length handling

### Actions
- Disabling unnecessary core features for performance

### Helpers
- Secure input sanitization utilities

## Concepts Covered
- add_action
- add_filter
- WordPress lifecycle hooks
- Security best practices
